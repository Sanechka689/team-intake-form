# AGENTS.md

## Repository expectations
- This repository is dedicated to Codex workspace organization and reusable Codex workflows.
- Keep changes minimal, explicit, and easy to audit.
- Do not restructure unrelated project code unless the task explicitly asks for it.
- A task is done when files are created or updated correctly and structure is verified.

## Standard workflow
- Read existing instructions first.
- Apply the smallest useful change.
- Verify resulting folder structure after edits.
